# Binary-Classification-for-Credit-Card-Fraud-Detection
## Training and Evaluating models for [credit card fraud detection](https://www.kaggle.com/datasets/mlg-ulb/creditcardfraud), a highly unbalanced dataset.

# Models Trained:
- Logistic Regresion
- Random Forest
- K Nearest Neighbors
- XGBoost

# Techniques Used:
- Stratified Split
- Polynomial Features
- SMOTE
